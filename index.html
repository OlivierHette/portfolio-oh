Groupomania

26 mai 2022

Groupomania est un projet que j'ai réalisé dans le cadre de ma formation.
C'est une API de réseau social d'entreprise.
Il permet à des membres d'une entreprise d'echanger entre eux si ils possèdent un compte. Ce reseau social est codé exclusivement en Javascript, avec React.js pour le Frontend et Node.js pour le Backend. La base de donnée est en SQL. Vous pouvez retrouver ce projet sur GitHub
